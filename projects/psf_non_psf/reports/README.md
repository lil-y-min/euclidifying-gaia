# PSF/non-PSF Reports (clean names)

This folder is the project-level report view with readable names (no date prefixes).
Artifacts are symlinks to canonical files under `report/model_decision/`.

## Sections
- `labeling/`: label creation + score calibration
- `gate/`: frozen gate variants + stability checks
- `experts_moe/`: specialist experiments + MoE runs + feature importance
- `latent_cvae/`: CVAE and latent analyses
- `decisions/`: decision notes and final reports
- `runbooks/`: operational runbooks

## Quick open
- Main report: `reports/decisions/recent_work_report.md`
- Direct shortcut: `RECENT_WORK_REPORT.md`
